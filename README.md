# Community Website - Local Test Version

This version does not use React Router, Babel, or any CDN JavaScript.
It is designed to work reliably in VS Code Live Server.

## How to run
1. Extract the zip
2. Open the folder in VS Code
3. Right-click `index.html`
4. Choose `Open with Live Server`

## Notes
- Home/About and Projects/Contact are still separated as two views
- Navigation switches views with plain JavaScript
- Includes 14 member cards
- Replace `members/member1.svg` ... `member14.svg` with real photos anytime
